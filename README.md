## 🚲 Cyclistic Bike-Share Analysis
**Case Study: How Does a Bike-Share Navigate Speedy Success?**

### 📋 Project Overview
The goal of this analysis is to identify how annual members and casual riders use Cyclistic bikes differently. These insights will help design a new marketing strategy to convert casual riders into annual members.

### 🛠️ Tools Used
* **SQL (BigQuery):** Data cleaning and large-scale aggregation of **5.7M+ rows**.
* **Python (Pandas, Seaborn, Matplotlib):** Exploratory Data Analysis (EDA) and visualization.

### 💡 Key Insights
* **Seasonal Trends:** Identified a **93% seasonal drop** in casual ridership during winter months.
* **Usage Patterns:** Annual members show steady commuting patterns, while casual riders peak during weekends and summer months.

### 📂 Files in this Project
* [View Analysis Notebook](cyclistic-case-study-capstone.ipynb)
Since the raw data is quite large, I have included three aggregated CSV files in this repository for quick review:
* **[annual_summary.csv](annual_summary.csv)**: Aggregated totals and average ride lengths by member type.
* **[day_of_the_week_usage.csv](day_of_the_week_usage.csv)**: Daily ridership volume to identify weekly trends.
* **[monthly_usage.csv](monthly_usage.csv)**: Monthly volume to visualize seasonal fluctuations.

## 📚 E-Learning Student Engagement Analysis
**Project Link:** [View the Interactive Tableau Dashboard](https://public.tableau.com/app/profile/luke.shofestall/vizzes)

### 📋 Project Overview
This business intelligence project focuses on analyzing user behavior, product utilization, and customer engagement metrics across an e-learning platform. By tracking how users interact with training modules, this analysis identifies key drop-off points and trends to help cross-functional teams optimize content delivery and improve course completion rates.

### 🛠️ Tools Used
* **SQL:** Data manipulation, filtering, and structuring student interaction logs.
* **Tableau:** Built a scalable, interactive dashboard to monitor core business metrics including completion milestones, user retention, and platform utilization.
* **Google Sheets / Excel:** Used for initial data validation and quick metric auditing.

### 💡 Key Insights & Strategic Impact
* **Product Utilization:** Mapped user journey milestones to pinpoint exact modules where engagement levels drop, providing actionable insights for content engineering teams.
* **Engagement Trends:** Segmented platform usage by user demographics and time-of-day trends to help customer success teams optimize targeted re-engagement campaigns.
* **Data-Informed Decisions:** Translated complex technical usage metrics into visual, strategic language for stakeholders to drive product improvements.
